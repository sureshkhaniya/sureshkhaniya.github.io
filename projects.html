World Association Learning Activity (Node.js / Express / EJS)

Project Type: Upper-Division Academic Application (Multi-Module)

Technical Stack: Node.js, Express.js, EJS, JavaScript, HTML/CSS

The World Association Learning Activity is an iterative Node.js application developed across multiple modules to reinforce user interface design, server-side routing, and application structure using Express and EJS. The project is intentionally built in stages and continues to evolve as additional functionality is introduced.

The application includes a user registration workflow where users self-enter personal and demographic data through dynamically rendered EJS forms. Additional user fields were identified through documentation review and integrated using appropriate form controls, including text inputs and list boxes, ensuring proper data submission via POST requests.

Express routers were implemented to handle both GET and POST operations. Browser developer tools were used to inspect network traffic and verify correct form submission behavior, ensuring all fields were transmitted accurately to the server.

The project also introduces role-based application design by distinguishing between administrative users and regular users. Administrative functionality includes report selection and result pages that simulate median values and top-performing results for selected educational demographics using dummy data.

While this module does not yet integrate a database, the application was architected to support future data persistence and querying in later modules.

Core Competencies: Express Routing (GET/POST), EJS Templating, Form Handling, UI Design, Role-Based Application Structure, Debugging with Developer Tools

Future Enhancements: Integrate database persistence, implement real statistical queries, and expand authentication and authorization mechanisms.
